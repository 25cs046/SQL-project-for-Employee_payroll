## Project Overview
A payroll management system to handle employee salaries, deductions, and department-wise reporting.

## Tables
- **departments** — stores department names
- **employees** — stores employee details linked to departments
- **payroll** — stores salary, deductions, and net pay

## Features
1. All employees with their department names (JOIN)
2. Net salary calculation (basic - deductions)
3. Highest paid employees (LIMIT)
4. Department-wise salary summary (GROUP BY)
5. Deduction percentage calculation

## Concepts Used
- JOIN, multi-table JOIN
- Mathematical calculations on columns
- GROUP BY with SUM and AVG
- LIMIT (top N results)
- Percentage calculation in SQL

SQLite
